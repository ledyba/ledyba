# Kaede Fujisaki / 藤咲かえで

今でもレトロな路面電車が街中を走る微睡町。夢と現実の曖昧なこの町で、この世界ではないどこかにあるという「希望」を探しながら、長い長い石の階段の上にある、町を一望できる高台で、古いおもちゃを扱う小さな店を営んでいる。

> この世界には、 希望だけがないからさ。
